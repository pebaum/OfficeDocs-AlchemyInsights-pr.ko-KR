# <a name="increase-the-size-of-a-shared-mailbox"></a>공유 사서함의 크기 늘리기

공유 사서함에 Exchange Online P2 라이선스 (E3 및 E5 라이선스에 포함 됨)를 할당 하 여 공유 사서함의 크기를 기본 크기인 50에서 100 GB로 늘릴 수 있습니다. [활성 사용자](https://portal.office.com/adminportal/home)의 공유 사서함에 P2 라이선스를 할당 하면 크기가 자동으로 15 분 내에 증가 됩니다.

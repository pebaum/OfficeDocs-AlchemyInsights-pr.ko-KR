# <a name="gatewaynotreachable-when-setting-credentials"></a>자격 증명을 설정할 때에 연결할 수 없음

데이터 원본에 대 한 자격 증명을 설정 하려고 하면 연결에 연결할 수 없습니다. 이는 오래 된 게이트웨이의 결과일 수 있습니다. 최신 게이트웨이를 설치 하 고 다시 시도 하세요.

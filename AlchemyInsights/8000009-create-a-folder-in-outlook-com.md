# <a name="create-a-folder"></a>폴더 만들기

1. 왼쪽 창에서 **폴더**옆에 있는 **새 폴더 만들기** 를 선택 합니다. 
    
2. 새 폴더 상자에 폴더의 이름을 입력 하 고 enter 키를 누릅니다.
    
자세한 내용은 [Outlook.com에서 폴더 만들기](https://go.microsoft.com/fwlink/p/?linkid=873114)를 참조 하십시오.

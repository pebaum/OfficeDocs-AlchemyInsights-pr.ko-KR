# <a name="how-to-delete-a-folder"></a>폴더를 삭제 하는 방법

1. 폴더 창에서 삭제할 폴더 또는 하위 폴더를 마우스 오른쪽 단추로 클릭 합니다.
2. **폴더 삭제**를 선택 합니다.
3. **확인** 을 선택 합니다.

자세한 내용은 [웹용 Outlook에서 메시지 폴더 작업](https://support.office.com/article/ae0f10d6-54e7-4f29-acd3-78cdc3fdcb9f)을 참조 하십시오.
